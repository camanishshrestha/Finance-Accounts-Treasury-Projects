## 🎯 Project Objective

USAID audits are among the **most rigorous donor audits** in the international development sector — governed by strict US government regulations including **2 CFR 200 (Uniform Administrative Requirements, Cost Principles, and Audit Requirements)**. The audit must answer: **"Were US taxpayer funds used in accordance with applicable US laws, regulations, and the terms of the award?"** As Team Leader/Team Member, I conducted USAID project audits with a specific focus on **cost allowability, cost allocability, and cost reasonableness** — the three pillars of USAID expenditure compliance.

## 🛠️ Step-by-Step: What I Did

---

### ✅ Step 1: Understanding USAID Compliance Framework

- Studied the **USAID regulatory and compliance framework** — significantly more complex than other donor audits:

#### 📋 Key USAID Regulations

| Regulation | Description |
|------------|-------------|
| **2 CFR 200** | Uniform Guidance — the master regulation governing all US federal awards covering administrative requirements, cost principles, and audit requirements |
| **ADS 303** | USAID Grants and Cooperative Agreements — specific USAID policies for award management |
| **ADS 591** | Financial Audits of USAID Contractors and Recipients |
| **FAR (Federal Acquisition Regulation)** | Governs USAID contracts (as opposed to grants/cooperative agreements) |
| **USAID Standard Provisions** | Mandatory and required-as-applicable provisions attached to every USAID award |

#### 📋 USAID Cost Principles (The "Three Tests")

| Test | Question | Implication |
|------|----------|-------------|
| **Allowability** | Is this cost type permitted under 2 CFR 200 and the award terms? | Certain costs are explicitly unallowable (entertainment, alcohol, lobbying, etc.) |
| **Allocability** | Is this cost properly chargeable to this specific award? | Costs must benefit the award they are charged to |
| **Reasonableness** | Is the amount reasonable for the goods/services received? | Costs must reflect what a prudent person would pay |

### ✅ Step 2: Audit Planning — USAID-Specific Approach

- Developed **audit plan** aligned with USAID audit requirements:

#### 🔹 Award Document Review
- Reviewed the **Award Agreement** (Cooperative Agreement / Grant / Contract):
  - Award number, period of performance, total obligated amount
  - Program description and objectives
  - Approved budget by cost category
  - Special conditions and restrictions
  - Cost-share / matching requirements (if any)
  - Reporting requirements
- Reviewed **USAID Standard Provisions** applicable to the award:
  - Mandatory provisions (apply to all awards)
  - Required-as-applicable provisions (apply based on award characteristics)

#### 🔹 Scope & Risk Definition
- Identified **key audit focus areas**:
  - Cost allowability per 2 CFR 200 Subpart E
  - Procurement standards per 2 CFR 200.317-327
  - Sub-recipient monitoring per 2 CFR 200.332
  - Cost-share verification (if applicable)
  - Property management
  - Internal controls
- Determined **audit sample** using risk-based and statistical sampling methods

---

### ✅ Step 3: Cost Allowability Testing

> This was the **core audit procedure** — testing each sampled expenditure against the **three cost tests**

#### 📋 Allowability Test Framework

| Check | Verification Performed |
|-------|----------------------|
| **2 CFR 200.403** | Cost is necessary and reasonable for performance of the award |
| **2 CFR 200.404** | Cost is allocable to the award based on relative benefits received |
| **2 CFR 200.405** | Cost conforms to award terms and conditions |
| **2 CFR 200.420-476** | Cost is not in the list of specifically **unallowable costs**: |
| | — Alcoholic beverages (§200.423) |
| | — Entertainment (§200.438) |
| | — Lobbying (§200.450) |
| | — Fundraising (§200.442) |
| | — Fines and penalties (§200.441) |
| | — Bad debts (§200.426) |
| | — Contingencies (§200.433) |
| | — Certain advertising/PR (§200.421) |
| **Award-Specific** | Cost is not prohibited by specific award terms |

#### 🔹 Transaction-Level Testing
- For each sampled transaction:
  - Verified the cost was **necessary** for project implementation
  - Verified the cost was **allocable** to the specific USAID award (not another project)
  - Verified the cost was **reasonable** — appropriate amount for goods/services
  - Checked against the **list of unallowable costs** per 2 CFR 200
  - Checked against **award-specific restrictions** in the agreement

---

### ✅ Step 4: Expenditure Vouching — Transaction Testing

- Performed detailed **vouching of sampled expenditure transactions**:

#### 📋 USAID Vouching Checklist

| Document | Verification |
|----------|-------------|
| **Invoice/Receipt** | Original invoice from vendor; complete details (vendor name, description, amount, date) |
| **Purchase Order/Contract** | PO/contract issued before goods/services received; authorized signatures |
| **Goods Received Note** | Evidence that goods were actually received; quantity and quality verified |
| **Service Completion Certificate** | For services — confirmation that services were satisfactorily completed |
| **Competitive Procurement Evidence** | Quotations, bid documents, evaluation report (based on threshold) |
| **Approval Documentation** | Expenditure approved by authorized personnel before payment |
| **Payment Evidence** | Bank transfer, check copy, or cash voucher with recipient signature |
| **Budget Line Alignment** | Cost charged to correct budget category as per approved budget |
| **NICRA Application** | If indirect costs charged — verified Negotiated Indirect Cost Rate Agreement (NICRA) rate applied correctly |

#### 🔹 Transaction Classification
- Flagged transactions as:
  - ✅ **Allowable** — fully compliant, adequately supported
  - ⚠️ **Questioned** — compliance concerns, may need to be refunded to USAID
  - ❌ **Unallowable** — clearly prohibited, must be refunded

---

### ✅ Step 5: Indirect Cost (NICRA) Verification

- Verified **indirect cost charges** — a unique aspect of USAID audits:

#### 📋 Indirect Cost Verification Framework

| Check | Details |
|-------|---------|
| **NICRA Agreement** | Verified organization has valid Negotiated Indirect Cost Rate Agreement with USAID or cognizant agency |
| **Rate Applied** | Verified indirect cost rate applied matches NICRA-approved rate |
| **Base Calculation** | Verified correct base used (Modified Total Direct Costs — MTDC or other approved base) |
| **Exclusions from Base** | Verified exclusions from MTDC base per 2 CFR 200.68: |
| | — Equipment |
| | — Capital expenditures |
| | — Sub-awards beyond first $25,000 |
| | — Participant support costs |
| **Computation Accuracy** | Recalculated: `Indirect Cost = MTDC × NICRA Rate` |

#### 🔹 Identified Issues
- **Over-claimed indirect costs** — incorrect rate or base used
- **Missing NICRA** — indirect costs charged without valid NICRA agreement

---

### ✅ Step 6: Procurement Compliance Audit

- Verified compliance with **USAID/2 CFR 200 procurement standards**:

#### 📋 2 CFR 200 Procurement Requirements by Threshold

| Threshold | Requirement |
|-----------|-------------|
| **Micro-Purchase (≤$10,000)** | Purchase without competitive quotations; reasonable price |
| **Small Purchase ($10,001–$250,000)** | Price or rate quotations from adequate number of qualified sources |
| **Sealed Bids (>$250,000)** | Formal advertising, sealed bids, public bid opening, award to lowest responsive bidder |
| **Competitive Proposals (>$250,000)** | RFP, evaluation based on price and other factors, negotiations permitted |
| **Non-Competitive (Sole Source)** | Only permitted under specific circumstances with documented justification |

#### 🔹 Procurement Verification Steps
- For each sampled procurement:
  - Verified **correct procurement method** applied based on value
  - Verified **competition** — adequate number of quotes/bids obtained
  - Verified **vendor selection** — based on documented evaluation criteria
  - Verified **conflict of interest** — no conflicts in selection process
  - Verified **debarment check** — vendor not on SAM.gov excluded parties list
  - Verified **contract/PO execution** — before goods/services delivered
  - Verified **reasonableness of price** — market comparison, historical pricing

#### 🔹 Identified Irregularities
- **Inadequate competition** — fewer quotations than required
- **Sole source without justification** — non-competitive procurement without documented reason
- **Contract splitting** — deliberately dividing purchases to avoid thresholds
- **Conflict of interest** — relationship between staff and selected vendor

### ✅ Step 7: Sub-Recipient Monitoring Audit

- Verified **sub-recipient monitoring** compliance per 2 CFR 200.332:

#### 📋 Sub-Recipient Monitoring Verification Framework

| Requirement | Verification |
|-------------|-------------|
| **Risk Assessment** | Prime conducted risk assessment of sub-recipient before sub-award |
| **Sub-Award Agreement** | Proper sub-award agreement executed with all required terms (FFATA, provisions flow-down) |
| **Monitoring Plan** | Prime has documented monitoring plan for sub-recipients |
| **Financial Monitoring** | Prime reviews sub-recipient financial reports, conducts site visits |
| **Programmatic Monitoring** | Prime verifies sub-recipient is achieving program objectives |
| **Audit Requirements** | Sub-recipients meeting single audit threshold ($750,000) obtained required audits |
| **Findings Follow-Up** | Prime follows up on sub-recipient audit findings and ensures corrective action |

- Reviewed a **sample of sub-recipient expenditure** — applying same allowability, allocability, reasonableness tests

---

### ✅ Step 8: Cost-Share Verification (If Applicable)

- If the award required **cost-share/matching contribution**, verified:

#### 🔹 Cost-Share Verification Checklist
- **Cost-share commitment** — required percentage per award agreement
- **Actual cost-share contributed** — compared against requirement
- **Cost-share eligibility** — contributions meet 2 CFR 200.306 criteria:
  - Verifiable from recipient's records
  - Not from other federal sources (unless permitted)
  - Necessary and reasonable
  - Allowable under 2 CFR 200 cost principles
  - Incurred within the award period
- **In-kind contributions** — properly valued (volunteer time, donated goods)
- **Documentation** — adequate records supporting cost-share claims

---

### ✅ Step 9: Property Management Review

- Verified **property management** for equipment purchased with USAID funds:

#### 📋 Property Management Verification

| Check | Verification |
|-------|-------------|
| **Property Records** | Maintained property records with description, serial number, acquisition date, cost, location, condition |
| **Physical Inventory** | Annual physical inventory conducted |
| **Control Procedures** | Adequate safeguards to prevent loss, damage, or theft |
| **Maintenance** | Equipment properly maintained |
| **Disposition** | Equipment disposed of per USAID requirements at end of award |
| **USAID Marking** | Equipment properly marked/branded as USAID-funded |

- Conducted **physical verification** of sample equipment — confirming existence, condition, usage

#### 📋 Internal Control Areas Assessed (COSO Framework)

|------|-----------------|

  - Financial reporting reliability
  - Compliance with award terms

  - Efficiency of operations

### ✅ Step 11: Audit Report Preparation

- Prepared **USAID-compliant audit report**:

#### 📋 Audit Report Components

| Section | Content |
|---------|---------|
| **Auditor's Report on Financial Statements** | Opinion on Schedule of Expenditure |
| **Auditor's Report on Internal Control** | Findings related to internal control deficiencies |
| **Auditor's Report on Compliance** | Findings related to non-compliance with award terms, 2 CFR 200 |
| **Schedule of Expenditure** | Audited expenditure statement by budget category |
| **Schedule of Findings & Questioned Costs** | Detailed findings with: |
| | — Condition (what was found) |
| | — Criteria (what should have been) |
| | — Cause (why it happened) |
| | — Effect (financial and compliance impact) |
| | — Questioned costs (amounts to be potentially refunded) |
| | — Recommendation |
| **Corrective Action Plan** | Management's response and corrective action plan |

#### 🔹 Questioned Costs Quantification
- Quantified **questioned costs** — amounts that must potentially be refunded to USAID:
  - **Ineligible costs** — clearly unallowable per 2 CFR 200
  - **Unsupported costs** — inadequate documentation to confirm allowability
- Submitted report to **USAID Mission** and **Office of Inspector General (OIG)** as required
