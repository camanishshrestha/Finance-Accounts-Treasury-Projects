## 🎯 Project Objective

Save the Children is one of the world's largest **child-focused international organizations** operating in over 100 countries. Auditing Save the Children programs requires not only **financial compliance verification** but also an understanding of **humanitarian program delivery, child safeguarding requirements, and multi-donor funding complexities** where a single program may receive funding from multiple donors — each with their own reporting and compliance rules. As Team Leader/Team Member, I conducted audit of Save the Children program expenditures — ensuring funds were used for **intended child-focused programs** with adequate controls and compliance.

> ⚠️ *Note: All program-specific financial data, beneficiary information, and audit findings remain strictly confidential. Only general audit methodologies and task descriptions are shared here.*

### ✅ Step 2: Audit Planning — Multi-Donor Program Complexity

- Developed **audit plan** addressing the unique complexities of Save the Children audits:

#### 📊 Key Audit Challenges

| Challenge | Description |
|-----------|-------------|
| **Multi-Donor Funding** | Single program funded by multiple donors — each dollar must be tracked to specific donor |
| **Cost Allocation** | Shared costs allocated across programs and donors — methodology must be fair and documented |
| **Field Office Operations** | Programs implemented through remote field offices — physical verification required |
| **Partner Expenditure** | Significant funds flow through implementing partners — partner controls must be assessed |
| **Humanitarian Spending** | Emergency response spending may have relaxed controls — higher fraud risk |
| **Child Safeguarding** | Compliance with child safeguarding policies — unique to child-focused organizations |

#### 🔹 Scope Definition
- Defined **audit scope**:
  - Specific programs/awards to be audited
  - Period covered
  - Field offices and partners included
  - Expenditure categories to be tested
- Selected **audit sample** using:
  - **Risk-based sampling** — high-value transactions, cash payments, emergency spending
  - **Random sampling** — representative coverage across expense categories
  - **Judgmental sampling** — unusual items, year-end transactions, new partners

### ✅ Step 3: Program Expenditure Verification

- Verified **program expenditure** against approved budgets and donor requirements:


#### 🔹 Variance Analysis
- Analyzed variances:

  - **Overspending** — exceeding approved budget without authorization

### ✅ Step 4: Transaction-Level Vouching

- Performed detailed **vouching of sampled expenditure transactions**:

#### 📋 Save the Children Vouching Checklist

| Check | Verification |
|-------|-------------|
| **Payment Voucher** | Properly filled, numbered, dated, with clear description |
| **Authorization** | Approved by authorized personnel per delegation of authority |
| **Supporting Documents** | Complete supporting documentation: |
| | — Vendor invoice/bill |
| | — Purchase order/contract |
| | — Goods received note / service completion |
| | — Quotations (competitive procurement) |
| | — Attendance sheets (trainings/workshops) |
| | — Participant lists with signatures |
| | — Event reports (training/workshop reports) |
| | — Travel authorization and travel report |
| **Donor Coding** | Expenditure correctly coded to specific program/donor/project code |
| **Budget Line** | Charged to correct budget category |
| **Accounting Entry** | Correctly recorded in accounting system (SUN/Agresso/SAP) |
| **Tax Compliance** | TDS deducted and remitted where applicable |
| **Receipt** | Official receipt/acknowledgment from payee |

#### 🔹 Identified Exceptions
- Identified:
  - **Unsupported expenditure** — missing or incomplete documentation
  - **Misallocated expenditure** — charged to wrong program or donor
  - **Duplicate payments** — same expense paid twice
  - **Ineligible expenditure** — costs not permitted under donor agreement

### ✅ Step 5: Training & Workshop Expenditure Audit

- **Training and workshop costs** constituted a significant portion of program expenditure — requiring specialized verification:

#### 📋 Training/Workshop Verification

| Document | Verification |
|----------|-------------|
| **Training Plan** | Training aligned with program objectives and approved annual plan |
| **Participant List** | Signed attendance sheet with participant names, organization, signature, date |
| **Participant Count** | Number of actual participants matches claimed participants |
| **Per Diem/Refreshment** | Per diem/refreshment costs reasonable; receipts available |
| **Venue Cost** | Venue invoice; reasonableness of venue selection |
| **Facilitator Fees** | Contract/TOR for facilitator; daily rate reasonable |
| **Training Materials** | Receipts for materials; materials relevant to training topic |
| **Training Report** | Post-training report documenting content, outcomes, feedback |
| **Photographs** | Photographs of training event as supporting evidence |

#### 🔹 Key Risk Areas
- **Inflated participant numbers** — claiming per diem for non-existent participants
- **Ghost participants** — names on attendance list but didn't actually attend
- **Excessive per diem/refreshment** — above organization or donor-approved rates
- **Unnecessary trainings** — trainings conducted without clear program justification

### ✅ Step 6: Field Office Audit & Site Visit

- Conducted **field office visits** to verify operations and expenditure at the field level:

#### 📋 Field Office Audit Activities

| Activity | Details |
|----------|---------|
| **Cash Management** | Reviewed petty cash management — cash count, cash book accuracy, replenishment process, cash ceiling compliance |
| **Bank Reconciliation** | Verified field office bank reconciliation — completeness, timeliness, outstanding items |
| **Advance Management** | Reviewed staff and partner advances — issuance, settlement timelines, aging analysis |
| **Asset Verification** | Physical verification of assets at field office — computers, vehicles, furniture, program supplies |
| **Inventory/Stock Check** | Physical count of program supplies (educational materials, health kits, nutrition supplies) |
| **Staff Verification** | Verified field office staff existence — matching payroll records with actual staff present |
| **Beneficiary Verification** | Sample verification of program beneficiaries — confirming actual beneficiaries exist and received program benefits |
| **Program Activity Verification** | Visited program implementation sites — schools, health posts, community centers — to verify reported activities |

- Documented field visit findings with **photographs and signed verification records**

### ✅ Step 7: Implementing Partner (Sub-Grantee) Audit

- Reviewed expenditure of **implementing partners** who received sub-grants from Save the Children:

#### 📋 Partner Expenditure Audit Process

| Step | Activity |
|------|----------|
| **1** | Obtained partner **financial reports** and supporting documents |
| **2** | Verified **partner expenditure** against approved sub-grant budget |
| **3** | Selected **sample transactions** from partner books and vouched |
| **4** | Verified **procurement compliance** at partner level |
| **5** | Reviewed **partner bank reconciliation** and cash management |
| **6** | Assessed **partner internal controls** — segregation of duties, authorization, record-keeping |
| **7** | Verified **partner asset management** — assets purchased with SCI funds |
| **8** | Reviewed **partner financial reports** submitted to SCI for accuracy |
| **9** | Identified **unsupported or ineligible expenditure** at partner level |
| **10** | Assessed **partner capacity** — ability to manage donor funds with adequate controls |

#### 🔹 Key Partner Audit Risks
- **Weak internal controls** — small NGOs/CBOs with limited financial management capacity
- **Inadequate documentation** — informal record-keeping practices
- **Commingling of funds** — SCI funds mixed with other organizational funds
- **Unauthorized expenditure** — partner spending outside approved budget
- **Late reporting** — delayed financial reports from partners

---

### ✅ Step 8: Cost Allocation Review

- Reviewed **cost allocation methodology** for shared costs across programs and donors:

#### 📋 Cost Allocation Verification

| Check | Details |
|-------|---------|
| **Allocation Methodology** | Documented cost allocation methodology exists and is consistently applied |
| **Allocation Basis** | Reasonable allocation basis used — staff time, direct cost ratio, beneficiary count, or other logical basis |
| **Shared Staff Costs** | Staff working on multiple programs — time sheets reviewed, time allocation to each program verified |
| **Shared Office Costs** | Rent, utilities, internet, security — allocated based on documented methodology |
| **Vehicle Costs** | Vehicle usage allocated based on log books or program usage records |
| **Consistency** | Same methodology applied consistently across periods — no cherry-picking of allocation methods |
| **Donor Agreement** | Allocation methodology compliant with each donor's specific requirements |

#### 🔹 Identified Issues
- **Unsupported allocation** — costs allocated without documented basis
- **Inconsistent methodology** — different methods applied in different periods
- **Over-allocation to specific donor** — disproportionate costs charged to one donor

---

### ✅ Step 9: Child Safeguarding Compliance Check

- Verified compliance with **Save the Children Child Safeguarding Policy** — a unique audit area specific to child-focused organizations:

#### 📋 Child Safeguarding Verification

| Check | Verification |
|-------|-------------|
| **Policy Awareness** | Staff aware of child safeguarding policy — signed acknowledgment on file |
| **Training** | All staff completed mandatory child safeguarding training |
| **Background Checks** | Background/police checks conducted for staff working with children |
| **Reporting Mechanism** | Whistleblowing/reporting mechanism available for safeguarding concerns |
| **Incident Register** | Safeguarding incident register maintained (if any incidents) |
| **Partner Compliance** | Implementing partners have child safeguarding policy in place |
| **Program Design** | Program activities designed with child safeguarding considerations |
| **Photographs/Media** | Photo consent obtained before photographing children; no identifying information published without consent |

> While this was not a traditional financial audit area, it was a **critical compliance requirement** unique to Save the Children audits

### ✅ Step 10: Stock & Inventory Verification

- Verified **program stocks and inventory** — particularly relevant for humanitarian and health programs:

#### 📋 Stock Verification Process

| Step | Activity |
|------|----------|
| **1** | Obtained **stock register** / inventory records |
| **2** | Selected **sample items** for physical count |
| **3** | Conducted **physical stock count** at warehouses, field offices, distribution points |
| **4** | Compared **physical count vs. book balance** — identified discrepancies |
| **5** | Reviewed **stock receipts** — goods received notes, donor/supplier delivery documents |
| **6** | Reviewed **stock issuance** — distribution records, beneficiary acknowledgments |
| **7** | Checked for **expired/damaged stock** — particularly medicines, nutritional supplements |
| **8** | Verified **storage conditions** — appropriate storage for sensitive items |
| **9** | Assessed **stock management controls** — access controls, dual custody, segregation |

#### 🔹 Typical Program Stocks Verified
- Educational materials (textbooks, stationery, learning kits)
- Health supplies (medicines, first aid kits, hygiene kits)
- Nutrition supplies (therapeutic food, supplements)
- Emergency supplies (tarpaulins, blankets, water purification)

---

### ✅ Step 11: Audit Report Preparation

- Prepared **audit report** in compliance with Save the Children audit reporting requirements:

#### 📋 Audit Report Structure

| Section | Content |
|---------|---------|
| **Auditor's Opinion** | Opinion on whether expenditure was fairly presented and incurred in accordance with program objectives and donor requirements |
| **Audited Financial Statement** | Statement of expenditure by budget category — audited figures |
| **Notes to Financial Statement** | Accounting policies, basis of preparation, exchange rates |
| **Findings** | Specific audit observations: |
| | 🔴 High priority — significant non-compliance or control failure |
| | 🟡 Medium priority — moderate risk findings |
| | 🟢 Low priority — minor observations and improvements |
| **Questioned Costs** | Ineligible or unsupported expenditure amounts |
| **Recommendations** | Specific, actionable recommendations for each finding |
| **Management Response** | Save the Children management's action plan for each finding |

#### 🔹 Key Findings Categories
- **Financial findings** — unsupported costs, misallocations, procurement non-compliance
- **Internal control findings** — segregation of duties gaps, cash management weaknesses
- **Partner findings** — partner-level documentation and control deficiencies
- **Compliance findings** — donor-specific requirement non-compliance
- **Asset management findings** — inventory discrepancies, asset register gaps

- Submitted audit report to **Save the Children Country Office** and **Regional Office** as required
