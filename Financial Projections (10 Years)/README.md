## 🎯 Project Objective

Develop a 10-year financial projection model for a debenture issuance to demonstrate:

- Debt servicing capability
- Principal repayment capacity
- Regulatory compliance (NRB, SEBON)

---

## 🛠️ Work Summary

### Base Setup
- Used audited financial statements as base year
- Analyzed historical trends (growth, profitability, ratios)

### Assumptions
- GDP, inflation, credit growth
- Interest rate outlook
- Regulatory environment (NRB/SEBON)

### Financial Statements Projection
- Balance Sheet (Assets, Liabilities, Equity)
- Income Statement (Interest income, expenses, NPAT)
- Cash Flow Statement (Operating, Investing, Financing)

### Key Analysis
- DSCR (Debt Service Coverage Ratio)
- Capital Adequacy Ratio (CAR)
- Key financial ratios (ROA, ROE, NIM, NPL, etc.)

### Scenario Analysis
- Base case
- Optimistic case
- Pessimistic case
- Stress scenario

### Model Validation
- Balance sheet balancing
- Cash flow reconciliation
- Formula integrity checks
- Scenario testing

---

## 📊 Key Outcomes

- Built full 10-year financial model
- Ensured DSCR > 1.0x across all scenarios
- Maintained regulatory CAR compliance
- Supported successful debenture issuance
- Approved by management and regulators
